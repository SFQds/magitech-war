# 流水线的早晨

> 纪元1342年春 · 铁锤城，第七工业区，铁砧重工主厂房

---

早晨六点，汽笛响了。

铁锤城第七工业区中央矗立着一座四米高的金属哨塔——水塔的外挂改装，联邦人称它为"早鸣塔"。塔身是铆接铁板的，焊口粗糙但结实得能扛住去年那场台风。塔体内藏着三颗脉晶共振块，每天早晨六点整，高压蒸汽从下方锅炉房推上来，强迫水晶达到共振频率临界点——爆发出来的声音介于蒸汽管风琴的冲击和轮船雾笛之间，七百赫兹，足以穿过任何墙壁、任何睡眠、任何刚才还在做梦自己不用上班的潜意识。

莉娜·齿轮在第七声汽笛时睁开眼。她的身体已经被训练成了七声闹钟——前六声在潜意识层被过滤掉了，第七声才会触发"起床"指令。这不是铁砧重工的培训。这是九年来住在第七工业区低级女工宿舍的自适应进化——她的窗户正对着早鸣塔，塔离窗口不到五十米。新搬来的女工第一周通常会在第一声汽笛时惊醒、尖叫、以为自己被鱼雷轰炸了、然后坐在床边哭。到第四周她们能睡过前五声。到赋闲两个月她们连汽笛都听不见了——永久听不见，噪声性听力损伤，铁砧重工福利部负责免费配助听器。

她从床头挂钩一把扯下昨天挂在那就没折的工服。六点三分——洗漱，头发卷进工帽（联邦安全规范要求长头发必须卷进工帽，防止被流水线齿轮绞入——这条规范是两年前一个没卷头发的女工被绞掉了半片头皮后才加进手册的。莉娜认识那个女工。她叫玛尔塔，没死，但走路时习惯性地往左侧偏，像在给什么人让道）。六点十分出门，从宿舍穿过第七区的铁架天桥——天桥底下是铁路货场，一排排的矿石车皮停在晨雾里，每个车皮上刷着"铁砧"字样——六点十五在北门打卡，六点二十分站到她的工位：三号流水线，左起第四个，代号"3-4"。标签就贴在流水线铁架边缘，一张防水胶膜，上面用帝国字母（联邦工业界沿用了帝国字母的标注传统，虽然两国几乎在其余一切事上都互不承认对方的格式——但字母不是政治，字母只是谁打赢了标准制定委员会）印着她的名字和工号。"莉娜·L·齿轮。LF-0441。"

她的岗位叫做"符文蚀刻质检"。工作内容是：流水线上的传送带以恒速推过来的每一个机甲关节铸件——刚浇铸成形、机床上刚走完第一道精削、还带着切削液的微温甘味——她戴上魔力放大器目镜，透过目镜的蓝色增强视野扫描每一道符文蚀刻的光洁度和连续性。如果符文线出现了断点——哪怕一个笔画的边缘模糊了，何等细小的断口她都抓得住——她用红色滑石笔把废品圈出来，扔进废品箱。废品拖出去重熔。每十分钟检查大约十二个铸件。每个铸件大致要注视不超过三秒——她在大脑后台预算的时间，不能停线——每条流水线上八个她这样级别的人。每一班运行十小时，十小时里她注视超过七百个铸件。每一个铸件会嵌入一台魔导机甲的关节部位，最终在战场上替某个她永远不会知道的士兵承受一发帝国法师的火焰冲击或水晶步枪的长距脉冲。

她十七岁进厂。母亲是个精瘦的不爱笑的女人，在矿场仓库里管了一辈子物料清册，手指甲缝里永远有洗不掉的紫色水晶粉。母亲送她到招工处那天说："你爸在矿井里待了二十五年。肺里的水晶粉尘搁在一块儿够铺一条小路了——能铺到邻居家门口。你去工厂。至少不用下井。你记住这个——你在工厂里日子怎么难过，井底下永远比你更难。"她记得自己说："嗯。至少。"她不知道自己当时在回答哪个部分，只是觉得说的任何话都没办法把"至少"这两个字拉成一句感恩的话。这些年她经常把那两个字从脑子里翻出来重新嚼一遍。九年了，"至少"还卡在牙缝里。不是恨。也不是认命。是在一个通常够不着其他选项的人生里，对最后一个勉强还可以接受的选项，保持警惕。

铁砧重工是全联邦数一数二的好工厂——他们说"好"不等于舒服，“好”是"有规矩"。规矩包括但不限于：每日不超过十小时（矿井十二小时起步），每两小时休息十五分钟（矿井不休息，渴了就对着井壁渗水舔一下），工伤厂方赔偿（矿井自行垫付且往往在第二年关门清册没了）。联邦其他工厂的工头们经常笑说铁砧是"贵妇厂"，意思是不舍得让工人受苦。莉娜不觉得自己像贵妇，但她确实觉得铁砧在试图做对的事。老费舍——他们的车间主任，秃头，姓费舍，但她脑子里永远先想到"费舍的话"——每个月会把工伤统计表贴在车间布告栏上，每个数字用红色粉笔圈好，旁边批着"要下降"、"上个月为什么高了三成"、"玛尔塔的事还没处理完"。不是做给工会看的——工会的检察员一年才来一次——是做给车间工人看的。你眼睛每天看到你同事的事故统计挂在你打卡的地方，你的手干活儿时会更慢更当心。老费舍说这是"最好的安全培训"。

但今天有点不同。

早晨的班组会上老费舍没有像平时那样先讲两分钟的安全提示。他站到班组前搓了很久的手——老费舍只有在坏消息或坏消息的预感面前才会搓手。然后他说："帝国宣了。"

全班组十二个人沉默了一分钟——不完全是在消化这句话的意思，而是汽笛外面的街区背景噪音太大了，人们需要一点时间把脑子从流水线噪音中抽出来。"什么叫宣了？"后排有人问。

"帝国皇帝在御前会议上宣布了——不再承认《翡翠和约》。他认为条约在帝国版本的法律体系下是'无效或已被实质废除'。就是上个月的事——你们在食堂有没有看报纸？"老费舍扫视了一圈，像是在问，也像是在提前推卸自己作为车间主任那部分没及时传新闻的责任。

没人举手说看了。他们中的一半不识字。另一半不读帝国新闻。"现在的问题不是要不要打，是打不打我们都已经接到军部的加急订单了——铁砧接了。从今晚起三号流水线全部改产机甲关节——不再是普通批次，T-01型专用。月产量翻倍。加班费按联邦战时标准算，双倍时薪。但十小时工作制——"他目光往上抬，落在告示牌木框里那张印着《联邦工业标准法》薪资条款的泛黄公印纸上，"——暂时取消。取消多久我还不知道。"

又是沉默。一个你不认识的名字能取消你习以为常的权利时，你没办法用沉默来讨价还价。

莉娜先开口了。"老费舍。我想问一个问题。"

"说。"

"这批关节最后是不是装到马库斯厂长那台泰坦上的？"

工人们中间有人笑了——是被紧张挤出来的短笑，不是真的觉得好笑。泰坦原型机。他们每一个人都摸过那台机器上的零件，因为每一个零件都要经过这个车间。但他们大多数没见过泰坦怎样在战场上前进——他们见的只是散布在各条流水线上的零零碎碎还没有组装成任何形状的关节、活塞、齿轮、管线、护甲片。一个人很难把"自己刚才质检过了三个铸件"和"那个开着冒紫烟的大铁家伙正在帝国边境扛下三枚火球"这两件事连在一起。它们之间不是逻辑关系，是别人不愿意发现的属于另一个阶级的历史连续性。

"我不知道。"老费舍说。他是那种很少说"我不知道"的人——车间主任在工人面前不说"我不知道"，这会腐蚀他对下一班产量目标的可信度。他顿了一顿又补了一句："我也不太确定厂长会觉得自己的腿重要还是子弹卡壳更烦人。"

"那我换个问法。"莉娜把工作手套从围裙口袋里拉出来，一点一点地往手上戴——在胶布料蹭过手指时她不用思考，这动作已经焊在了肌肉记忆里。"我们这个月加班加点了。熬了夜，人累了——累了以后视线模糊，质检漏检率是会上升的。如果上升了——出厂时这批泰坦关节的不合格率比平时高——最后是谁负责签署终检出库单的？"

老费舍那双用来搓自己秃头的手停在半途。"我。跟平时一样，我来签。每天产量长多少倍也好，工时排多少小时也好，我签。因为我不签没人敢签；我不签的生产记录不是你晚上通宵出品的铸件纪录，是零。质量标准不动。明早进货的关节基准数就要在这上面直接给你画红线——超过了任何规定偏差，我不签字。铁砧重工东西，不合格就不是铁砧重工的东西。

莉娜把护目镜压到额头上，开了流水线开关。传送带嗡嗡启动，第一个新铸的关节滑过来——热的，刚出铸模，黄铜色的金属表面把车间顶棚的日光灯反射成一小块揉碎的太阳。她低头看进目镜里。关节面盆上的符文蚀刻走线精准得没有一丝要超出公差的痕迹。每一条线都是甜釉凝固前最后一秒的状态——现在它被这么刻进了铁砧重工的金属关节里，会被装进T-01，在某个下午的乱炮里被帝国法师的某种她叫不出名字的高温咒语斜射击中。运气好的话它能撑过去，运气不好它连同那个操作机甲的军需官一起绞碎在碎石路面的泥水坑里。这些她全都知道。但这些都不由她负责。她现在的工作是把目镜下每一个拐角看过，确认它还活着——还够着火线另一头去当护甲。

她心想：至少出厂的时候它是好的。它将来在哪里、被什么事打碎——那些，都不是今天早晨我能管的事。**

她拿起下一个关节。
