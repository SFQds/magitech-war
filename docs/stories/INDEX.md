# 魔导工业革命 — 故事集

> 本故事集是 `CODEX.md` 的叙事补充。以短篇小说形式，讲述从第一台水晶引擎到纪元1342年游戏开始的完整历史，以及这个世界中每一个重要人物的命运。
>
> 共21篇。

---

## 故事目录

### 旧世界的终结

- [第一台引擎](./001-first-engine.md) — 纪元1203年冬，老铁砧在铁匠铺的废弃矿井旁第一次启动了水晶引擎
- [总督的选择](./012-victor-ironhand.md) — 维克托·铁腕，被判处终身流放，在孤岛上种了十年土豆。联邦打赢后回来签了第二个名字
- [铁砧女士](./013-anna-anvil.md) — 安娜·铁砧，第二代。她把父亲的铁匠铺变成了联邦最大军工厂，发明了流水线和标准件

### 战争与战士

- [碎脊之夜](./002-shattered-spine.md) — 纪元1246年深秋，帝国三个法师军团在碎脊山口被矿工用水晶步枪从山体内部击退
- [翡翠和约](./014-jade-accord.md) — 纪元1248年春，玉港城海风楼。帝国和联邦打不下去了但没人敢先开口说"和谈"。翡翠的烤鱿鱼解决了这个问题

### 帝国的暮色

- [帝国的早晨](./003-imperial-morning.md) — 纪元1340年秋，帝国元老塞维鲁喝走私热炉煮的柠檬水，给议会起草一份他明知会被塞进档案室最暗抽屉的备忘录

### 英雄列传

- [伊莎贝尔·默库里](./004-isabelle-mercury.md) — 被除名的十二冠继承人，在漏风铁匠铺里发明了默库里合金
- [塞巴斯蒂安·柯格斯沃](./005-sebastian-cogsworth.md) — 撬下圣殿符石装到引擎上的帝国子爵之子
- [艾琳·灰烬](./006-eileen-ash.md) — 矿工女儿，共鸣体质，安全共鸣器发明者
- [马库斯·铁砧](./007-marcus-anvil.md) — 铁砧重工第三代厂长，开漏紫烟的泰坦原型机上前线

### 齿轮背后的人

- [影笔](./015-corvin-shadowquill.md) — 科尔文·影笔，被帝国流放的法师，艾琳的导师。流放原因是一本论文的第二章
- [守门人](./016-temple-high-priest.md) — 圣殿祭司长，无名老人。在地下室守了四十年，等一个能读懂奥古斯都手稿的人
- [暗流](./017-the-smuggling-route.md) — 一块水晶从霜脊矿井到帝国法师塔的走私全旅程，经手六个人，没有一个人说谎，但没有一句全是真的
- [嬗变](./018-alchemist-origin.md) — 梅茜·硅砂，炼金协会创始人。她的异常档案夹除了数据还有一只她自己刻的水母章
- [山礼](./019-mountain-ritual.md) — 纪元1307年冬至，艾纳尔赤脚站在北山峰顶一整夜。山说可以
- [首次会议](./020-void-first-meeting.md) — 纪元1324年初冬，七个陌生人钻进废弃矿场。没有成立宣言——虚空研究院从这一天开始存在

### 世界的裂缝

- [翡翠港的市集](./008-jade-market.md) — 卡林·渡口在玉港城中央市场的一天，三个客人，一个比一个危险
- [地下的心跳](./009-heartbeat-underground.md) — 纪元1337年冬，老矿工戈尔曼在地下关灯后看到了不该存在的光
- [霜脊的沉默](./010-frostridge-silence.md) — 山王对帝国使节说"踩在我的山上——很不礼貌"

### 凡人的日子

- [流水线的早晨](./011-assembly-morning.md) — 纪元1342年春，莉娜·齿轮在铁砧重工三号流水线加班
- [阴影下的街](./021-shadow-street.md) — 纪元1340年，帝国下城区。一条每天只能照到一个时辰阳光的街上，五个人的同一天
