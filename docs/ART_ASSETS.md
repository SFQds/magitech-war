## 一、资源规格

| 项目 | 规范 |
|------|------|
| 瓦片尺寸 | 32 × 32 px |
| 单位精灵 | 64 × 64 px（载具/机甲 96 × 96） |
| 建筑精灵 | 96 × 96 px（指挥中心 128 × 128） |
| 格式 | PNG（透明背景） |
| 风格 | 2D 俯视/等距，像素风或干净平涂 |
| 调色 | 阵营主色 + 水晶紫(#9b59b6) + 工业橙(#e67e22) |

## 八、占位符方案

在没有美术资源时，代码中已使用以下占位方案：

| 对象 | 占位方式 | 代码位置 |
|------|----------|----------|
| 地形 | Graphics.fillRect 纯色块 | `GameScene.renderTiles()` |
| 单位 | Graphics.Rectangle 小色块 | `GameScene.create()` |
| 建筑 | Graphics.Rectangle 较大色块 | — |
| 资源点 | 紫色半透明方块 | `GameScene.placeInitialResources()` |
| 迷雾 | 黑色/半透明黑矩形覆盖 | `GameScene.renderFogOfWar()` |
| UI | 系统字体 + Graphics 矩形 | HUDScene / UI组件 |

> 占位方案可支撑开发和测试，美术资源可分批替换。
